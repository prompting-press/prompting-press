frozen into its own versioned docs site at snapshot time.

You are viewing the **`next`** (in-development) docs, which track the `main`
branch and have no frozen release entry yet. To read a release's changelog,
pick a released version from the version switcher at the top of the sidebar,
then open its **Changelog** page — or browse the full
[GitHub releases](https://github.com/prompting-press/prompting-press/releases).
